# Rest_Api_proj
# Simple REST API using Python HTTP Server

A minimal REST API built using Python's built-in http.server module without any external framework.

This project demonstrates core backend concepts such as:
- Handling HTTP requests manually
- Creating REST endpoints
- JSON request and response handling
- Basic routing
- API testing using Postman

---

## 🚀 Features

- GET API endpoint
- POST API endpoint
- JSON request handling
- JSON response handling
- Basic error handling
- No external framework used
- Lightweight API server

---

## 🛠 Tech Stack

- Python 3
- http.server (built-in library)
- JSON
- REST API concepts

Tools:
- Postman (API testing)
- Git
- GitHub
